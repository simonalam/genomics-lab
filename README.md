# genomics-lab

Submission for Simon Alam and Miky Asmare.

This lab explores gene selection and regularization for leukemia classification using the Golub et al. (Science, 1999) dataset of ~7,000 genes across 72 patients. We compare ordinary least squares regression against cross-validated LASSO to demonstrate how regularization handles high-dimensional genomic data, and discuss the implications of these methods for precision health.
